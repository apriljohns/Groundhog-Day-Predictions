# Groundhog Prognosticators

### Created by April Johns, Josh Daniels, and Molly Murphy, ERHS 535 Fall 2025

## How to explore the data

Click through the tabs in the dashboard to see the locations of the prognosticators, an interactive leader board with their overall accuracy, and their accuracy when adjusted for seasonal consistency. Methods and documentation can be found on the last tab.

## Data Sources:

### Groundhog data: 

[Tidy Tuesday](https://github.com/rfordatascience/tidytuesday/blob/main/data/2024/2024-01-30/readme.md)

The original data and the code for extracting and cleaning the data from [grounhog-day.com](https://groundhog-day.com/) was from Tidy Tuesday, groundhog data was re-extracted on December 3rd, 2025 to include 2025 prognostications. The original code is accessible [here](https://github.com/rfordatascience/tidytuesday/blob/main/data/2024/2024-01-30/readme.md?plain=1).
